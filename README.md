# 📜 AI TaskBot

An AI-powered Task Management Assistant built using:

- Streamlit
- LangChain
- LangGraph
- SQLite
- Groq LLM

The assistant can create, update, read, and delete tasks using natural language.


# 🚀 Features

- Create Tasks
- View Tasks
- Update Task Status
- Delete Tasks
- AI-Powered SQL Interaction
- Streamlit Chat UI
- SQLite Database
- Conversation Memory


# 🛠️ Tech Stack

- Python
- Streamlit
- LangChain
- LangGraph
- SQLite
- Groq API


# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/your-username/AI-TaskBot.git
cd AI-TaskBot
````

## Install Dependencies

```bash
pip install -r requirements.txt
```

## Add Environment Variables

Create `.env`

```env
GROQ_API_KEY=your_api_key
```

---

# ▶️ Run Project

```bash
streamlit run app.py
```
